DOSULDOSOFT



QUIENES SOMOS

Douldosoft se quiere enfocar en que sea una tienda de computadores, o accesorios de este mismo como los son, Mouses, Auriculares, Teclados, Procesadores, Etc. Para así dar una evolución, en cuanto a las tiendas virtuales y llevar a cabo en este proyecto, cosas que sean mejoradas por decirlo así, queremos plantear nuevos avances en cuanto a la compra virtual.

OBJETIVOS GENERALES

Poder cumplir con nuestras expectativas con el desarrollo de nuestro programa, para poder generar un avance a las tiendas virtuales.

OBJETIVOS ESPECIFICOS

Desarrollar nuestro programa al completo. Crear una base datos que se una al aplicativo web como de escritorio esto nos ayudara a guardar información al aplicativo web y de escritorio. Crear un aplicativo web haciendo que cualquier persona que tenga internet puede ingresar a nuestra tienda virtual hay podrá comprar. Mediante ala base de datos los empleados y el proveedor la podrán manejar con permiso del administrador del sistema
